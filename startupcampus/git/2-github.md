# Github 풀리퀘 절차
1. Master branch를 clone  
2. 마스터로부터 새로운 브랜치를 생성
    - 브랜치 이름은 구현할 기능 이름으로 지정
3. add/commit/push
4. pull request 생성
    - 베이스는 마스터로 지정
5. 리뷰어 지정
6. 리뷰어: 리뷰 및 리뷰 메시지 작성
7. 작업자: 리뷰 내용 보고 커밋/푸시
8. 6번과 7번을 반복 후 merge
